# Visual Graphs of Philosophy

This project explores the history of philosophy using data analysis and visualization techniques. The project uses philosophical text data to identify patterns, relationships, and connections between different philosophical ideas and thinkers.

## Project Overview

The analysis is performed using Python in a Jupyter Notebook. The philosophical dataset is processed and visualized to create meaningful graphs that help understand relationships within the history of philosophy.

## Files in This Repository

* `visual-graphs-of-philosophy.ipynb` – Jupyter Notebook containing the complete analysis and visualizations.
* `phil_nlp.csv` – Dataset used for the analysis.
* `README.md` – Project documentation.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* Data Analysis
* Data Visualization
* Natural Language Processing (NLP)

## Dataset

The project uses the **History of Philosophy** dataset, which contains philosophical text data used for analysis and visualization.

## How to Run

1. Download or clone this repository.
2. Make sure Python and Jupyter Notebook are installed.
3. Open `visual-graphs-of-philosophy.ipynb`.
4. Make sure `phil_nlp.csv` is in the same folder as the notebook.
5. Run the notebook cells step by step.

## Project Goal

The goal of this project is to use computational methods and visual graphs to explore philosophical texts and discover meaningful patterns and relationships in the history of philosophy.
